Test Token Modal

Instrucciones

Esta página permite probar el modal de alerta de token. Haz clic en el botón para simular un problema con el token de autenticación.

window.tokenHasIssues = true;

Cuando esta variable global se establece en true, el componente TokenAlert detectará el problema y mostrará el modal.

Cómo funciona

El componente TokenAlert verifica periódicamente si la variable global window.tokenHasIssues está establecida en true. Cuando detecta este valor, muestra el modal de alerta.

En una aplicación real, esta variable podría establecerse cuando:

  • Una solicitud API devuelve un error de autenticación
  • El token ha expirado
  • Se detecta un problema de seguridad
